## setup

**Clone the project**

```shell
git clone https://github.com/allanvobraun/JsCanvasConnect4.git
```

**install dependencies via yarn**
```shell
yarn install
```
**build and run localy**
```shell
yarn run hot
```
Then open your browser at http://localhost:9000/ and enjoy!
